# Чат-бот

Простой чат-бот на клиентском JavaScript, который умеет считать числа и выводить
результат прямо в окно чата. Серверной части нет — всё работает в браузере.

## Возможности

- Адаптивная вёрстка под любые экраны, на десктопе ширина чата ограничена 700px
- Лента сообщений идёт снизу вверх
- Кнопка отправки неактивна (серая), пока поле пустое, и становится жёлтой после ввода текста
- Размер баблика подстраивается под количество текста
- Высота поля ввода не меняется при большом объёме текста
- При ответе бота показывается анимация в виде троеточия

## Команды

- `/start` — начало общения. Бот спросит имя. Если ввести что-то другое до старта,
  бот попросит ввести `/start`
- `/name: Вася` — бот поздоровается и предложит ввести числа
- `/number: 7, 9` — задаём числа через запятую, после чего бот спросит действие
- `-`, `+`, `*`, `/` — выбор действия, бот посчитает и вернёт результат
- `/stop` — завершение общения

Если введена неизвестная команда, бот ответит: «Я не понимаю, введите другую команду!».

## Запуск

Проект не требует сборки и зависимостей. Достаточно открыть `index.html` в браузере.

Либо запустить локальный статический сервер из папки проекта:

```bash
python -m http.server 8000
```

После этого открыть в браузере `http://localhost:8000`.
